# Facebook Clone

A basic Facebook-like webpage built using HTML and CSS.

## Features
- Responsive design for modern devices.
- Custom CSS styling.
- Static front-end structure.

## Folder Structure
- **`assets/`**: Contains images and media assets.
- **`css/`**: Stylesheets for the webpage.
- **`index.html`**: Main HTML file for the project.
